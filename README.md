# favalid

Validator Framework for javascripts.

[![CircleCI](https://circleci.com/gh/akito0107/favalid.svg?style=svg)](https://circleci.com/gh/akito0107/favalid)

## Getting Started

### Prerequisites
- Node.js v8+
- npm or yarn

### Installing
```
$ npm install favalid
```
